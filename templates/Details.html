Project Details

The aim of this project was to create an interactive way in which a user can enter a tweet and find the sentiment of the tweet automatically.This was created by using Natural Language Processing tools and Machine Learning models.This project is based on vaccine related tweets only but can be re-trained to find sentiment on any keyword of our choice.
